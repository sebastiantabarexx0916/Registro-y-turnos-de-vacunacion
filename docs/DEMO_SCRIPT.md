Guion de demostración (3-5 minutos)

1) Introducción rápida (30s)
   - Nombre del proyecto y objetivo.

2) Preparar entorno (15s)
   - Indicar que el proyecto está compilado (`ant`) o abrir NetBeans.

3) Flujo mínimo (2-3 min)
   - Crear un `Paciente` de ejemplo (nombre, documento, fecha nacimiento).
   - Agendar un turno para el paciente (centro, fecha/hora, vacuna).
   - Mostrar lista de turnos programados.
   - Registrar vacunación: seleccionar turno, seleccionar `Lote` y `Vacunador`, confirmar registro.
   - Generar y mostrar un `Reporte` sencillo (número de vacunaciones por fecha).

4) Cierre (30s)
   - Señalar características adicionales: gestión de usuarios, notificaciones, ISO/25010 mapping.
   - Preguntas y siguiente pasos.
